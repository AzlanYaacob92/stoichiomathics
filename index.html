How it works: moles n = m ÷ M. For aA + bB → … the amount of B needed to use up all the A is n(A) × b/a. If that's more than you have, B runs out first (B is limiting); if it's less, A runs out first. The reactant left over is the one in excess.
Relative atomic masses follow common data-booklet values (H = 1.0, O = 16.0, Cl = 35.5, Cu = 63.5 …). State symbols and spectator H⁺/OH⁻ are omitted. Always re-check against your own mark scheme.
